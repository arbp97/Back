St. Patrick Homebanking API

Repository Main page Documentation

> /account/auth - POST body content response

Send a valid email and password - responds with sent email & access token (stp-token).

email: example@email.com
password: password application/json email: example@email.com
token: TOKEN error: error

> /account/find - POST body content header response

Send a valid email and token - responds with Account object.

email: example@email.com stp-token: TOKEN application/json Account error: error

> /account/transfer - POST body content header response

Make a money transfer between two valid accounts. Origin and destiny refers to the CCI code.

origin: CCI
destiny: CCI
motive: some motive
amount: > 0 stp-token: TOKEN application/json status: success error: error

> /accounts/user - POST body content header response

Send a DNI & valid token - responds with all Accounts with the same owner.

dni: DNI stp-token: TOKEN application/json Account [] error: error

> /user/find - POST body content header response

Send a user's DNI & valid token - responds with User object.

dni: 12345678 stp-token: TOKEN application/json User error: error

> /transaction/find - POST body content header response

Send a valid transaction ID & token - responds with Transaction object.

transactionId: >= 0 stp-token: TOKEN application/json Transaction error: error

> /transactions/account - POST body content header response

Send a valid account CCI & token - responds with all Transactions having this account as the origin or destiny.

cci: CCI stp-token: TOKEN application/json Transaction [] error: error
