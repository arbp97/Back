St. Patrick Homebanking API

Repository Main page Documentation

> /account/auth - POST

Send a valid email and password - receive sent email & access token (stp-token)

body: {
"email": "example@mail.com",
"password": "************"
} type: "application/json" response: {
"email": "example@mail.com",
"token": "****************"
} || { "error": error }

> /account/find - POST

Send a valid email and token - receive Account object

body: { "email": "example@mail.com" } header: { "stp-token": "***************" } type: "application/json" response: { Account } || { "error": error }

> /account/transfer - POST

Make a money transfer between two valid accounts. Origin and destiny refers to the CCI code.

body: {
"origin": "000000000000000",
"destiny": "111111111111111",
"motive": "insert example motive"
"amount": 666
} header: { "stp-token": "***************" } type: "application/json" response: { "status": "success" }|| { "error": error }
